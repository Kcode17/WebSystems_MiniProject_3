W3C, HTML & CSS

W3C

The World Wide Web Consortium (W3C) is the main international standards organization for the World Wide Web. Founded in 1994 and currently led by Tim Berners-Lee, the consortium is made up of member organizations that maintain full-time staff working together in the development of standards for the World Wide Web. As of 21 October 2019, W3C had 443 members. W3C also engages in education and outreach, develops software and serves as an open forum for discussion about the Web.


HTML

HTML - Hyper Text Markup Language is a standard for language for creating web pages. It describes the structure of a web page. HTML elements have a starting tag with some content and an ending tag. (Ex: <p> This is a paragraph </p> )

Following things can be done using HTML

  • Make web pages with headings, text, tables, lists, photos, etc.
  • Navigate to other webpages via hypertext links, at the click of a button.
  • Design forms for making transactions with remote services for searching information, book tickets, ordering products etc.
  • With HTML, we can include spread-sheets, video clips, sound clips on web pages.
