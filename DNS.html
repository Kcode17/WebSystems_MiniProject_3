Domain Name System

What are Domain Name System?

Domain Name Systems is like a phonebook for the web. Users access websites using URLs like www.facebook.com, www.google.com etc. Web browsers work with IP addresses. Domain name system has a mapping with the domain name and its corresponding ip. Basically translates domain name to ip address.

Every device connect to internet has a unique ip address through which other devices can locate and interact. IP address are numbers which are hard to remember for humans, so DNS eliminates this step of remembering IPs.
